# AstridKJ.github.io

Personal website for Astrid KJ, built from the Academic Pages Jekyll template and hosted with GitHub Pages.

## Site Sections

- Home
- Research
- Publications
- Photography
- Arts
- CV

## Editing Content

- Site-wide identity and sidebar links: `_config.yml`
- Top navigation: `_data/navigation.yml`
- Main pages: `_pages/`
- Publication entries: `_publications/`
- Photography gallery entries: `_photography/`
- Arts entries: `_arts/`
- Images: `images/`
- Downloadable files such as CV PDFs: `files/`

## Local Preview

Install Ruby dependencies, then run:

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

The site will preview at `http://localhost:4000`.
